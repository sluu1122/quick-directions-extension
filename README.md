# Quick Directions

A Chrome extension that lets you highlight any address or location name on a webpage, right-click, and instantly get Google Maps directions from one of your saved starting points.

[Install from the Chrome Web Store](https://chromewebstore.google.com/detail/quick-directions/pghihakmbhdmhjepfijfdjnabfpioihg)

## Features

- Highlight any text on a webpage and right-click to get directions
- Save multiple starting locations (home, work, etc.)
- Optional "Search on Map" to search highlighted text directly on Google Maps
- Drag to reorder your saved locations
- Inline editing of saved locations

## Usage

1. Click the extension icon or go to **Settings** to add your starting locations
2. Highlight any address or place name on a webpage
3. Right-click and choose a starting location from the **Quick Directions** menu
4. Google Maps opens with directions from your saved location to the highlighted text

## Privacy

This extension stores your saved locations locally in Chrome's sync storage. No data is sent to any external servers. See [PRIVACY.md](PRIVACY.md) for details.
